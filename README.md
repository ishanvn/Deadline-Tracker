# Queue.

Crisp coordination. Zero missed targets. Absolute focus.

A lightweight, local-storage priority queue engineered specifically for tracking computer science and engineering coursework, labs, and project deadlines. 

## 🚀 Features
* **Local Storage Persistence:** Assignments are saved directly to the browser—no remote database required.
* **Chronological Auto-Sorting:** The underlying algorithm dynamically sorts array objects to push the most urgent deadlines to the top of the queue.
* **Zero-API Calendar Bridge:** One-click integration generates dynamic Google Calendar events using URL parameters.
* **Custom UI/UX:** Hardware-accelerated CSS glitch effects, custom keyframe animations, and a highly optimized dark-mode aesthetic.
